# Modules

LangChain provides standard, extendable interfaces and external integrations for
the following modules, listed from least to most complex:

**[Model I/O](/modules/model_io/models/models)**  
Interface with language models.

**[Retrieval](/modules/retrieval/retrieval)**  
Interface with application-specific data.

**[Chains](/modules/chains/chains)**  
Construct sequences of calls.

**[Agents]()**  
Let chains choose which tools to use given high-level directives.

**[Memory](/modules/memory/memory)**  
Persist application state between runs of a chain.

**[Callbacks]()**  
Log and stream intermediate steps of any chain.
