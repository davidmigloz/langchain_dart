# OllamaEmbeddings

```dart
final embeddings = OllamaEmbeddings(model: 'llama3.2');
const text = 'This is a test document.';
final res = await embeddings.embedQuery(text);
// or for multiple documents
// final res = await embeddings.embedDocuments([text]);
```
